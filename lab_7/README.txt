синхронное - последовательное
ассинхронное - одновременное

wg sync.WaitGroup

WaitGroup - группа (пачка) горутин

wg.Add(n) - добавить n горутин в пачку
wg.Done() |внутри горутины| - убирает горутину из пачки
wg.Wait() |внутри *глобальной горутины* (напр. main)| - блокирует горутину пока все пачка не станет пустой

Как на самом деле (проще)
у wg есть счетчик
Add(n) - счетчик прибавляется на n
Done() - счетчик убавляется на 1
Wait() - "ставит на паузу" поток, до тех пор пока счетчик != 0

HTTP - формат передачи данных в виде текста (гипертекста) без шифрования
HTTPS - с шифрованием ("с паролем(сертификатом)")
URI - уникальный индентификатор ресурса (путь к ресурсу/полное имя ресурса)
URL - протокол+домен("имя сайта")+uri
основные метод ы
GET: получить ресурс
POST: создать ресурс
PUT: обновить ресурс
DELETE: удалить ресурс
